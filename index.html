<char-counter>

A simple, configurable Polymer component for adding a character counter to your form fields.

About

This is a simple drop-in web component that lets you add a character counter to your web form through a <char-counter /> tag. Check out the examples below to see what sort of customizations the component offers, then check out the documentation for further info.
